Документация по tgbot-телеграм бот
Инструкция по запуску
Локальный запуск
Требования:
Python 3.14
Установленный SQLite3
Установка зависимостей:
Создайте виртуальное окружение и установите необходимые библиотеки:
python -m venv venv
.\venv\Scripts\activate
pip install pyTelegramBotAPI google-genai
Запуск:
python bot.py
Отчет об использовании ИИ (Согласно п. 5.4)
В проекте использована генеративная модель для обработки сообщений, не являющихся командами
Используемая модель:
Google Gemini 2.5 Flash.
Инструменты:
Библиотека Python: google-genai.
Системный промпт (System Instruction):
"Ты — ИИ-помощник бота Aqmola Start. Твоя задача: помогать пользователю находить стартапы, специалистов, команды и события региона. Отвечай кратко, по делу, без воды. Понимай запросы (поиск, рекомендации, вопросы об экосистеме) и направляй к нужной информации. Не выдумывай данные — если нет информации, говори об этом и предлагай ближайшие варианты. ВАЖНО: Не используй смайлики. Не используй Markdown-форматирование. Пиши только простой текст."
Область применения:
ИИ используется исключительно для ответов на вопросы в свободной форме (FAQ, советы, навигация), когда запрос пользователя не соответствует жестким командам меню.
